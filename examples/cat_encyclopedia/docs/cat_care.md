# 猫咪养护指南

## 日常喂养

### 主食选择
猫咪是纯肉食动物，需要高蛋白饮食。

#### 猫粮类型
1. **干粮**
   - 优点：方便储存、有助清洁牙齿
   - 缺点：含水量低（仅10%）
   - 适合：成猫日常主食

2. **湿粮（罐头）**
   - 优点：含水量高（70-80%）、适口性好
   - 缺点：开封后易变质、价格较高
   - 适合：补充水分、增加食欲

3. **冻干**
   - 优点：营养保留完整、轻便
   - 缺点：价格昂贵
   - 适合：零食或辅食

#### 猫粮选择标准
- 蛋白质含量 ≥ 30%
- 谷物含量低（最好无谷物）
- 必须含有牛磺酸（猫咪无法自行合成）
- 避免含有人工色素和防腐剂

### 喂食频率
- **幼猫（2-6月龄）**：每天3-4次，少食多餐
- **成猫（1-7岁）**：每天2次
- **老年猫（7岁以上）**：每天2-3次，少食多餐

### 饮水
- 每天提供新鲜清水
- 每日饮水量标准：40-60ml/kg体重
- 使用饮水机可以增加猫咪饮水量

### 禁忌食物 ⚠️
- **巧克力**：含有可可碱，对猫有毒
- **洋葱、大蒜**：破坏红细胞，导致贫血
- **葡萄/葡萄干**：可能导致肾衰竭
- **牛奶**：大多数猫乳糖不耐受
- **咖啡因/茶**：刺激神经系统
- **酒精**：对肝脏和神经系统有害

## 日常清洁

### 梳毛
- **短毛猫**：每周1-2次
- **长毛猫**：每天1次
- **换毛季**（春秋）：增加频率
- **工具**：针梳、排梳、去毛刷

### 洗澡
- **频率**：每1-3个月1次（不需要太频繁）
- **水温**：38-39℃
- **洗毛用品**：必须用猫专用沐浴露
- **禁忌**：不要用人用沐浴露（pH值不同）

### 剪指甲
- **频率**：每2周1次
- **注意**：只剪透明部分，不要剪到血线
- **工具**：猫专用指甲剪

### 清洁耳朵
- **频率**：每周检查1次
- **方法**：用棉签轻轻擦拭外耳道
- **注意**：如有黑色分泌物可能是耳螨

### 刷牙
- **频率**：每周2-3次
- **工具**：猫专用牙刷和牙膏
- **注意**：不要用人的牙膏（含氟对猫有害）

### 猫砂盆清洁
- **每天**：铲屎1-2次
- **每2周**：彻底更换猫砂并清洗猫砂盆
- **猫砂盆数量**：猫数量 + 1（多猫家庭）

## 猫砂选择

### 膨润土猫砂
- **优点**：结团好、除臭力强、猫咪喜欢脚感
- **缺点**：粉尘大、重量大、不能冲马桶
- **适合**：追求性价比、猫咪不挑的家庭

### 豆腐猫砂
- **优点**：可冲马桶、粉尘少、环保
- **缺点**：结团一般、易受潮、价格稍高
- **适合**：公寓饲养、追求便利的家庭

### 水晶猫砂
- **优点**：吸水性强、使用时间长
- **缺点**：脚感差、无法冲马桶、价格高
- **适合**：出差频繁、不常在家的人

### 木屑猫砂
- **优点**：环保、吸水性好、可冲马桶
- **缺点**：木屑味重、容易散开
- **适合**：环保意识强的家庭

## 疫苗与驱虫

### 猫三联疫苗
预防三种疾病：猫瘟、猫鼻支、猫杯状病毒

**接种时间表**：
- 8周龄：第一针猫三联
- 12周龄：第二针猫三联
- 16周龄：第三针猫三联 + 狂犬疫苗
- 之后：每年加强1次

### 狂犬疫苗
- **接种时间**：3月龄以上
- **加强频率**：每年1次或每3年1次（根据疫苗品牌）

### 驱虫
**内驱（体内寄生虫）**：
- **频率**：每3个月1次
- **药物**：吡喹酮等
- **注意**：不出门的猫也需要驱虫

**外驱（体外寄生虫）**：
- **频率**：每1个月1次
- **药物**：塞拉菌素、氟雷拉纳等
- **注意**：滴在猫咪后颈部（舔不到的地方）

## 绝育手术

### 最佳时间
6-8月龄（第一次发情前）

### 母猫绝育
- **好处**：
  - 降低子宫蓄脓风险
  - 降低乳腺肿瘤风险
  - 减少发情期困扰
- **手术**：卵巢子宫摘除

### 公猫绝育
- **好处**：
  - 减少喷尿标记行为
  - 减少打架斗殴
  - 降低走失风险
  - 减少睾丸疾病
- **手术**：睾丸摘除

### 术前准备
- 禁食8小时（防止麻醉呕吐）
- 健康检查（血常规等）

### 术后护理
- 佩戴伊丽莎白圈（防止舔伤口）
- 限制活动7天
- 保持伤口干燥
- 观察是否有感染迹象

## 环境丰容

### 猫咪需要的环境要素

1. **垂直空间**
   - 猫爬架：满足攀爬天性
   - 窗台/高处：提供观察点
   - 猫抓板：磨爪、标记领地

2. **躲藏空间**
   - 猫窝、纸箱：提供安全感
   - 安静的角落：应对压力

3. **互动玩具**
   - 逗猫棒：模拟猎物运动
   - 激光笔：注意不要照射眼睛
   - 猫薄荷：增加乐趣（部分猫无反应）

4. **观鸟位置**
   - 窗台设置观察点
   - 可以提供鸟类喂食器吸引鸟类

### 多猫家庭建议
- **猫砂盆数量**：每只猫一个 + 1个备用
- **分开喂食区域**：避免食物竞争
- **提供足够的垂直空间**：减少冲突
- **轮换玩具**：保持新鲜感

## 日常互动

### 互动频率
- 每天15-30分钟
- 分多次进行（猫咪注意力集中时间短）

### 互动好处
- 增进感情
- 消耗精力，减少破坏行为
- 保持健康体重
- 减少压力和焦虑

### 常见互动方式
1. **逗猫棒**：模拟老鼠或鸟的运动
2. **激光笔**：追逐光点（结束时给零食作为"捕获"）
3. **猫爬架**：攀爬、跳跃
4. **纸箱**：猫永远喜欢纸箱！
5. **藏食游戏**：把零食藏在各处让猫寻找
