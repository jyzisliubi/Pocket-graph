# 电影知识图谱示例数据集

包含 50+ 部经典电影的知识图谱数据，涵盖电影、导演、演员、类型等实体。

## 数据概览

- **实体类型**：电影、导演、演员、类型、上映年份
- **关系类型**：导演、主演、类型、评分、上映年份、国家
- **三元组数量**：约 200 条

## 使用方式

### 1. 构建索引
```bash
# 设置数据路径
export POCKET_DATA_PATH=examples/movie_kg/data/movie_knowledge_graph_triples.txt

# 构建索引
cd PocketGraphRAG
python build_index.py
```

### 2. 启动 Web UI
```bash
python webapp.py
```
